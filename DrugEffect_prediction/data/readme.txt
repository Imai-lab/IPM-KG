The knowledge graph consists of DrugBank, Reactome, Gene Ontology Annotation, and The Comparative Toxicogenomics Database, each available from the following data.
DrugBank : Target Drug-UniProt Links data obtained from HP ( https://go.drugbank.com/ )
Reactome: Reaction pathway data expressed in BioPAX level 3 from HP (https://reactome.org/)
Gene Ontology Annotation: EBI Gene Ontology Annotation Database for Homo sapiens from HP (https://geneontology.org/docs/go-annotations/ )
The Comparative Toxicogenomics Database : Data obtained from Phenotype(GO)-Disease Networks of HP (https://ctdbase.org )
